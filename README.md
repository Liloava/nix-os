# NixOS Configuration

This repository contains my personal NixOS configurations for managing and deploying my systems, including my **PC** and **laptop**. It is structured to make it easy to maintain, update, and reproduce my system environments using [NixOS](https://github.com/NixOS).

## Features

- **Declarative system configuration** for multiple machines
- **Modular setup** for easy customization and reuse
- **Version-controlled** for tracking changes and rollback
- **Reproducible builds** for consistent environments
